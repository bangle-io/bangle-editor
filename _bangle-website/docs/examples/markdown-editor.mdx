Each of the [core component](/docs/api/core#components) of Bangle comes baked in with markdown shortcuts (ofcourse, you can disable them if you want).

The example below we see how you to parse Markdown content and how Bangle displays various components.

<VanillaCodeExample
  filePath={filePath}
  language="js"
  createEditor={createEditor}
  onEditorReady={(editor) => {
    return (
      <button
        onClick={() => {
          const mdText = serializeMarkdown(editor);
          const blob = new Blob([mdText], { type: 'text/plain' });
          window.open(URL.createObjectURL(blob));
        }}
      >
        Click here to see the Markdown serialization
      </button>
    );
  }}
/>
