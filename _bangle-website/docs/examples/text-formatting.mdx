Try some of the following things:

- Press `Ctrl-s` to start typing in superscript or `Ctrl-d` for subscript, press it again to stop.
- Please note you will need to switch off (by pressing the keyboard shortcut) one first, to use other.
- Try copy pasting superscript text from your favourite source.
