## Todo Lists

In this example we use the [TodoList component](/docs/api/core#todolist-component) to allow creation of todo lists.

Try some of the following:

- On a new line, type `[ ]` followed by a space to create a todo List.
- Press `Tab` to indent and `Shift-Tab` to unindent.
- Press `Ctrl-Enter` (mac) or `Ctrl-I` (PC) to mark as task done.
- `Ctrl-Shift-7` also converts a paragraph into a todo list.
- Drag the todo square to move reorder things by priority.
